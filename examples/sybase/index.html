Bookmark database





Welcome to the bookmarks database. This application provides a centralised and shared bookmark repository for your World-Wide Web bookmarks. It has been implemented using Sybase as the database for storage of the bookmarks and PHP/FI as the gateway between www and sybase.

The demo concept is taken from w3-msql.
